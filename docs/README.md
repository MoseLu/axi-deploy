将 axi-deploy 根目录的文档文件按功能分类整理，提高项目结构的清晰度和可维护性。

### 整理效果
1. **结构清晰** - 根目录只保留核心文件，文档按功能分类

3. **用户友好** - 新用户可以通过文档索引快速找到所需信息
4. **扩展性好** - 新增文档类型有对应的分类目录，便于后续文档的添加和管理
